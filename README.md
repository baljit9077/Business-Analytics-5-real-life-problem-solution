This coursework applies a range of business analytics techniques to solve real-world problems and support effective decision-making. It is divided into four main problems, each focusing on a different analytical approach: decision analysis, forecasting, optimisation, and simulation.

In Problem 1, decision analysis is used to evaluate strategies for a shipping company facing potential courier strikes. The Expected Monetary Value (EMV) approach identifies a probability threshold at which delaying shipments becomes more cost-effective than continuing normal operations. Bayesian analysis is then applied to incorporate predictive data, improving decision accuracy by updating strike probabilities. Additionally, the Expected Value of Information (EVI) is calculated to measure the benefit of having better information. The findings are summarised in a managerial report, which provides recommendations on when to delay shipments and when to proceed with operations.

Problem 2 focuses on developing a statistical forecasting model to predict customer arrivals and improve workforce planning. A regression model is initially built using multiple variables and is later refined by selecting only statistically significant predictors, such as day of the week, paydays, and holidays. The final model demonstrates strong predictive performance and helps explain variations in customer demand. A consulting report outlines how the model can be used to optimise staffing decisions and enhance operational efficiency.

In Problem 3, linear programming is used to maximise profit for a manufacturing company producing two products under labour constraints. The model determines optimal production quantities while considering available hours in different departments. Sensitivity analysis is conducted to evaluate the value of additional labour, highlighting which departments benefit most from overtime. This supports better resource allocation and cost-effective decision-making.

Problem 4 uses simulation to analyse production policies over a 52-week period. By running multiple iterations, the model accounts for uncertainty in demand and production. The results identify an optimal inventory threshold that minimises total costs.

Overall, the coursework demonstrates how data-driven techniques can improve decision-making, optimise resources, and support strategic business planning.
